# Usage Guide

## Login
- Default user: `admin`
- Password: `password`

## Features
1. Notes Manager ✏️
   - Add, update, delete notes
   - Export notes as CSV

2. Job Tracker 💼
   - Save job applications
   - Mark status: Applied / Interview / Rejected

3. Admin Dashboard 🔐
   - Secure access with JWT
   - LocalStorage token
